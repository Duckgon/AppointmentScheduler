3. From there, you have access to edit the customer table, edit the appointment tablem, view the reports, or log back out.

Additional Report Description: The additional report displays the number of appointments that have already passed the current date and time. It takes the end meeting times of each appointment and compares it to the current date and time. If the current date time is after the end meeting time, then the meeting is counted.

MySQL Connector driver version number: mysql-connector-java-8.0.25g
